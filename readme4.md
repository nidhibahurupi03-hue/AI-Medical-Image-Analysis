# AI Medical Image Analysis

This project detects pneumonia from chest X-ray images using Deep Learning.

## Features
- Image classification (Normal vs Pneumonia)
- CNN model using TensorFlow
- Flask web interface for prediction

## How to Run

1. Install requirements
pip install -r requirements4.txt

2. Train model
python main4.py

3. Run web app
python app4.py
